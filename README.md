### Задание 1. Кеширование 

Приведите примеры проблем, которые может решить кеширование. 

### Решение задания 1

```
Пример.
Чтобы отрисовать личный кабинет посетителя сайта, 
системе нужно собрать данные из 10 разных таблиц: 
профиль, последние заказы, бонусы, уведомления и т.д.

После первого входа пользователя агрегированные данные 
для его кабинета (например, user_dashboard_456) сохраняются в 
Redis на короткое время , как пример 5 минут.

Как итог пока посититель сайта ходит по личному кабинету, 
данные подгружаются мгновенно из кэша , нет необходимости 
нагружать базы данных. 

Посетитель сайта не ожидает долго ответа
от сервера а сразу же получает отрисованную страницу.
```
